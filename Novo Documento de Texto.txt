## Fluxograma ##

*** Diagrama de representação esquemática  de um processo ***

-------

| processo | preparação    | dados   |
| -------- | ------------- | ------- |
| decisão  | base de dados | display |

| terminal | vários documentos | documento |
| -------- | ----------------- | --------- |
|          | entrada manual    |           |





Não existe padrão, o fluxograma tem que ser claro.



Variável: objeto frequentemente localizado na memória.

Espaço na memória destinada a um dado que é alterado na execução do algoritmo.
